Karma

Image

display ( x, y )

displays the image (previously loaded).

load ( filename )

Loads the specified image file.

spriteImage ( filename, character, start, end ) *

Loads a bunch of image files. The 'character' in 'filename' is changed from start to end (inclusively)

Example: 
		//set up the environment
		var k=$.karma( "#karma-main" );
		//creates the canvas element
		k.size( 200, 200 );
		//creates the img element 
		var img = k.Image();
		//loads the img
		img.load("images/banana.png");
		//displays the img
		img.display( 0, 0 );
